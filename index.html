Seagoatvision

SeaGoatVision is a vision server to develop and execute filter on different media. The software is entirely written in Python and the filters can be written in both C++ or Python.

SeaGoatVision

Description

SeaGoatVision is a vision server to execute filter on picture.

  • Manage multiple client
  • Run filterchain
  • Manage multiple source
  • Give environnement for debug and create filter
